\section*{Lessons Learned The Hard Way}

\begin{enumerate}[leftmargin=*,topsep=0pt]
\item \textbf{Simple beats clever:} Every "clever" solution created two new problems
\item \textbf{Legacy code exists for a reason:} Usually because it actually works
\item \textbf{Perfect is the enemy of shipped:} My perfectly architected systems often never launched
\item \textbf{Documentation lies:} Especially when I wrote it 6 months ago
\item \textbf{Estimates are fiction:} Whatever I estimate, multiply by 3, add 2 weeks
\item \textbf{The user doesn't care:} About your beautiful architecture or clean code
\item \textbf{Meetings multiply:} The more senior you get, the less code you write
\item \textbf{Technology doesn't matter:} Business problems do, tech is just a tool
\end{enumerate}
