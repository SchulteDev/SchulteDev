\section{Core Learnings from Two Decades of Tech}

\begin{itemize}[leftmargin=*,topsep=2pt,itemsep=1pt]

\item Microservices multiply complexity, not reduce it
\item The cloud is just someone else's computer you pay more for
\item Legacy code is any code written more than 6 months ago
\item ``It works on my machine'' is not a valid deployment strategy
\item Documentation is like exercise: everyone agrees it's important, nobody does it
\item The temporary solution becomes the permanent architecture
\item There's no problem that can't be made worse with enough abstraction
\end{itemize}
