\subsection{October 2021 — December 2024: Cloud Architect at Union Investment}

\textbf{The Humbling:} Spent six months building a microfrontend framework that was more complex than the monolith it replaced. \textbf{The Redemption Arc:} Eventually built a working platform, but only after learning that "stateless" doesn't mean "thoughtless" and that event-driven architecture requires someone to actually think about what events mean.
